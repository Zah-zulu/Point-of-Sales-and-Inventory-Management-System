Point of Sales (POS) and Inventory Management System 
## Overview
This is a Point of Sales (POS) with integrated inventory management , built in Visual Studio and uses SQL Server for data management purposes. 
It simulates a real-world retail operation , allowing users to manage products, record sales, and generate reports (still not integrated for now)
The system was developed as part of an acedemic project and demonstrates practical application of database management, business logic, and system design.

## Features 
- User Login and Authentication
- Product and Inventory Management (add, update, remove products)
- sales recording and transaction tracking
- reporting functionality ( daily sales, and stock levels)
- role based control for different user types (Manager, Cashier)

## Technologies Used
-**Programming Language:**C#
-**Databases:**SQL Server Management Studio 
-**Tools:** Visual Studio, Git/GitHub, Power BI
